# Q-note

> 一个超级方便的浏览器拓展笔记本，自动通过域名分类。

## 快捷键

* 建议打开 chrome://extensions/shortcuts 设置快捷键为 `CMD + n` 全局监听调起。这样只要连续按两下 `CMD + n` 就可以快速添加笔记。
* `CMD/ALT/WIN + enter` ： 在添加笔记输入框输入完毕时直接添加。
* `CMD/ALT/WIN + n` ： 添加笔记。
